# ALLAN implementation for Indaco LegeAI

### Deployment
```
conda create -n allan1 tensorflow-gpu scikit-learn gensim
conda install -c pytorch pytorch
```


### Dev 
```
conda create -n allan_dev tensorflow-gpu=2.3 tensorflow=2.3=mkl_py38h1fcfbd6_0 tensorflow-hub gensim spyder==4.2.5 pandas matplotlib psutil tqdm shapely seaborn scikit-learn dropbox ipykernel=6.2.0
conda install -c pytorch pytorch==1.8
```
